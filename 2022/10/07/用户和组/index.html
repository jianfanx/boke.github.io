linux用户和组管理

用户

初级用户权限

r read w write x execute

添加

使用 useradd (用户名) 创建新用户

-d 指定主目录

-g 指定组

-G 指定附加组

-s 指定shell文件

-u 指定用户ID

删除

使用userdel(用户名)

-r 删除相关文件

修改

用户文件目录（/home/下）vi编辑文件

usermod

-l 锁定用户

-u 解锁用户

密码

添加用户密码是 passwd (用户名)

-l 锁定用户

-u 解锁用户

删除修改

查看文件 /etc/shadow 或 /etc/password

8个字段

- 用户名
- 密码
- 用户ID
- 组ID
- 备注及全称
- 用户主目录
- 用户账号过期市场
- 警告时长

查看文件字段

主要文件

/etc/login.defs

组管理

组管理文件夹目录

/etc/group

添加

groupadd

-g 指定组ID

删除

groupdel

修改

groupmod

-n 重新命名

-g 更改组ID

查看

tail 查看文件尾

head 查看文件头

ls 列出文件夹
